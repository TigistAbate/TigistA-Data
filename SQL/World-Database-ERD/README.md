# World Database ERD

## Overview

This project was completed during my Data Technician Bootcamp using MySQL Workbench.

## Objective

To design and explore a relational database by creating an Entity Relationship Diagram (ERD) that shows the relationships between tables.

## Skills Demonstrated

- MySQL Workbench
- Database design
- Entity Relationship Diagrams (ERDs)
- Primary keys
- Foreign keys
- One-to-many relationships
- Relational database modelling

## Project Features

- Designed a relational database model
- Created relationships between tables
- Used primary and foreign keys
- Visualised the database structure

## Outcome

This project improved my understanding of relational database design and how tables are connected using keys and relationships.
