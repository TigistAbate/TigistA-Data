## Project 1 – Data Cleaning and Analysis

### Dataset

Retail Sales Dataset

### Description

This dataset contains retail sales information, including customer details, product categories, quantities, prices and sales values. It was used to practise cleaning, organising and analysing data in Excel.

### Organisation

Retail business

### Why is this important?

Retail businesses use sales data to monitor performance, identify trends and support business decisions.

### Tools Used

- Microsoft Excel

### What I Did

- Converted the dataset into an Excel Table.
- Cleaned and organised the data.
- Used formulas and functions.
- Sorted and filtered records.
- Created PivotTables.
- Produced charts to summarise the data.

### Key Findings

- PivotTables summarised sales by county and product.
- The SWITCH function categorised sales volumes into meaningful groups.
