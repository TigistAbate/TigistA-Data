# Student Performance Analysis

## Overview

This project was completed during my Level 3 Data Technician Skills Bootcamp using Microsoft Excel. The aim was to analyse student performance by calculating average marks and highest scores, identifying the best-performing students, and highlighting key results using Excel tools.

## Objective

To analyse student marks in English, Mathematics and Science using Excel formulas, sorting, filtering and conditional formatting to identify student performance.

## Skills Demonstrated

- Microsoft Excel
- Data analysis
- Data organisation
- AVERAGE function
- MAX function
- Sorting and filtering
- Conditional formatting
## Organisation Type

This type of analysis could be used by a school, college or educational institution to monitor student performance and identify areas where additional academic support may be needed.
By analysing student performance, teachers can identify high-achieving students as well as those who may require additional support. This helps improve decision-making and supports better educational outcomes.

The dataset contains student names and their marks in English, Mathematics and Science. I used this dataset to calculate average marks, identify each student's highest score, and compare overall student performance.

## What I Did

- Organised the student marks dataset into an Excel table for easier analysis.
- Applied sorting and filtering to identify the highest-performing students in each subject.
- Used the `AVERAGE` function to calculate the average mark for each student.
- Used the `MAX` function to determine each student's highest subject score.
- Sorted the data by average mark to identify the top-performing student in the class.
- Sorted the data by highest score to compare students' best individual results.
- Applied conditional formatting to highlight the highest and lowest average scores, making the results easier to interpret.
- Reviewed the completed analysis to compare student performance across the class.

## Key Findings

- Calculating average marks provided a clear measure of each student's overall performance across all subjects, making comparisons more meaningful.

- Using the `MAX` function highlighted each student's highest subject score, helping identify their strongest area of performance.

- Sorting and filtering the data made it easy to identify the highest-performing students and compare results efficiently.

- Conditional formatting improved the readability of the dataset by visually highlighting the highest and lowest average scores, making key insights easier to identify at a glance.
