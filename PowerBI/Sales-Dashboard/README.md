# Adventure Works Sales Dashboard

## Overview

This project was completed during my Data Technician Bootcamp using Microsoft Power BI Desktop.

## Objective

To design an interactive sales dashboard that analyses sales performance across different countries, product categories, and financial years.

## Skills Demonstrated

- Microsoft Power BI Desktop
- Dashboard design
- Data visualisation
- Interactive reports
- Sales analysis
- Bar charts
- Column charts
- Line charts
- Slicers and filters

## Dashboard Features

- Sales by country
- Sales by product category
- Profit analysis
- Quantity sold by category
- Interactive filtering

## Outcome

This project improved my ability to create interactive dashboards and communicate business insights using Power BI.
